USER: Represents the individuals who are registered on the ShopEZ platform. Users can browse products, search and filter items, add products to the cart, place orders, track order status, and provide reviews for purchased products.

ADMIN: Represents the authority responsible for managing the platform. Admin can add, update, activate or deactivate products, manage categories, control user access, and update order and payment statuses.

PRODUCTS: Represents a collection of all the products available on the platform. It stores product details such as name, description, category, price, stock, images, featured status, and availability status.

CATEGORIES: This collection stores all product categories used to organize products into different sections on the platform.

CART: This module stores the products added by users before checkout. In ShopEZ, the cart is managed using localStorage and differentiates items based on the user session.

ORDERS: This collection stores all the orders placed by users on the platform. It includes order details such as order number, purchased products, shipping address, payment method, order status, and payment status.

REVIEWS: This collection stores product reviews submitted by verified buyers.

PAYMENTS: This collection maintains payment transaction details including payment method, payment status, and transaction reference.
